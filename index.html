What are html elements?

The starting and ending tag in html together are known as a html element. These are the building blocks of html which defines structure of the page.Some commonly used tags are headings - h1 to h6, paragraph - p, image - img, anchor - a, etc...

Note : There are html elements without closing tag. These are also known as self-closing tags. For ex, line brake - br, horizontal rule - hr etc...

What are Semantic & Non-semantic elements?

Semantic : Elements that have meaning or a purpose to the structure of html.Example, navagation - nav is used to specify certain navigation links in a page.

Non-semantic : Elements with no meaning just for representation.Example, div is used as block container, span is used as inline container for styling.

What are html attributes?

They provide additional information about html elements. Always specified in the starting tag. Example, attribute-name = "attribute-value" - src = "icon.jpg" i.e, src is an attribute of the img element. Attributes may vary from element to element.
